##Temporal Supervenience##

###Example: Standard weak individual supervenience:
A-properties weakly supervene on B-properties if and only if for any possible world w and any individuals x and y in w, if x and y are B-indiscernible in w, then they are A-indiscernible in w.

or,

A weakly supervenes on B if and only if necessarily, if anything x has some property F in A, then there is at least one property G in B such that x has G, and everything that has G has F.

###Diachronic supervenience claims:
__Type 1)__ No \`A\`-difference at time \`t'\` without \`B\`-difference at time \`t\`.

__Type 2)__ No \`A\`-difference from time \`t\` to time \`t'\` without \`B\`-difference from time \`t\` to time \`t'\`.

####Diachronic weak individual supervenience, type 1
\`A\`-properties weakly supervene on \`B\`-properties if and only if for any possible world \`w\` and any individuals \`x\` and \`y\` in \`w\`, if \`x\` and \`y\` are \`B\`-indiscernible at time \`t\` in \`w\`, then they are \`A\`-indiscernible in \`w\` at time \`t'\`.

Temporal operators are defined as:
- \`bbbP\`: it has at some time been the case
- \`bbbF\`: it will at some time be the case

\` squareAAxAAGinA[bbbF(Gx)->EEHinB(bbbP(Hx)^^AAy(bbbP(Hy)->bbbF(Gy)))] \`

\`A\` diachronically weakly supervenes on \`B\` if and only if necessarily, if anything \`x\` has some property \`G\` in \`A\` _at some future time_, then there is at least one property \`H\` in \`B\` such that \`x\` has \`H\` _at some past time_, and everything that has \`H\` _in the past_ has \`G\` _in the future_.

Notice that \`bbbP(Hx)\` and \`bbbP(Hy)\` need not be simultaneous. Similarly, \`bbbF(Gx)\` and \`bbbF(Gy)\` need not be simultaneous either.
